SudokuSolver

Overview:
This is a Sudoku puzzle solver created with Python. Two libraries were used in the creation of
this tool: tkinter and numpy. Tkinter was used to create the GUI for the tool, while numpy was
used in the actual Sudoku solution calculations. Recursion was used in order for this program to
actively solve puzzles.

How to Run:
With all the necessary files, run the following from the SudokuSolver directory:  python main.py
From there, follow the prompts on the GUI!
